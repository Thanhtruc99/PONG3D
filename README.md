# PONG3D
Game pong3d tunnel

## Description

Ce jeu 3D est une implémentation simple d'un environnement en forme de tunnel avec une balle et des panneaux. 

## Objectifs

Les principaux objectifs de ce projet sont de démontrer les concepts de base de la programmation graphique 3D à l'aide d'OpenGL/GLUT, de la détection de collision et des mécanismes de jeu simples.

## Fonctionnalités

- Mouvement du panneau contrôlé par le joueur à l'aide de la souris.
- Mouvement de la balle et détection de collision.
- Système de notation.
- Règles de jeu simples et commandes conviviales.


## Comment le récupérer
Avec terminal
Vous pouvez cloner ce dépôt en utilisant la commande suivante :

git clone https://github.com/Thanhtruc99/PONG3D.git
make
./PONG3D
